# **Шпаргалка по работе в консоле**

```Ls``` 	вывести содержимое директории
```Ls -a``` 	содержимое директории со скрытыми файлами
```Cd /имя_папки```	cменить директорию
```Pwd```	Показать текущую директорию
```Mkdir``` 	Создать директорию
```Mkdir -p   в/ы/ф/л```	Создать структуру директорий
```Touch имя_файла.расширение```	Создать файл
```Cp что_копируем куда_копируем/```	Копировать файл
```pbcopy < ~/.ssh/id_ed25519.pub```	Копировать в буфер обмена
```mv ~/Desktop/Privet/fail.txt ~/Desktop/Dva/fail.txt```	Перемещение файлов
```cat```        только текстовые файлы	прочитать файл
```Rm название_файла```	Удалить файл
```Rmdir название_директории```	Удалить директорию
```Rm -r название_диреткории```	Удалить директорию с содержимым
